# 🧠 Logica 2

Exercícios de Lógica com Python - Nível Avançado. Continuação da série de desafios para aprofundar seus conhecimentos em programação e algoritmos.

## 📋 Descrição

Este repositório é a **segunda parte** da série de exercícios de lógica com Python. Aqui você encontrará desafios mais complexos e avançados, desenvolvendo habilidades em estruturas de dados mais sofisticadas, algoritmos otimizados e padrões de resolução de problemas mais elaborados.

## 🎯 Objetivo

- Aprofundar conceitos de programação avançada
- Explorar estruturas de dados complexas
- Desenvolver algoritmos eficientes
- Praticar resolução de problemas desafiadores
- Preparar-se para entrevistas técnicas

## 🛠️ Tecnologias

- **Linguagem**: Python 3.x
- **Tipo**: Exercícios Educacionais Avançados

logica-2/

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/taallyr/logica-2.git
cd logica-2
```

2. Execute os exercícios:
```bash
python nome_do_exercicio.py
```

## 💡 Temas Cobertos

- Estruturas de Dados Avançadas (Pilhas, Filas, Árvores, Grafos)
- Algoritmos de Ordenação e Busca
- Recursão e Programação Dinâmica
- Análise de Complexidade (Big O)
- Manipulação de Strings Avançada
- Tratamento de Exceções
- Arquivos e I/O
- E muito mais...
